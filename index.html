    🕹️ Planner Controls

    Basic Mapping

    • Left Click Map: Place waypoint
    • Right Click Waypoint: Open context menu
    • Drag Map: Pan view

    Drawing Setup

    • Start Drawing: Right-click waypoint -> Select mode
    • Anchoring: Shapes automatically anchor to the selected waypoint.
    • Backspace / Delete: Detach shape from waypoint so it follows mouse freely.

    Drawing Controls (Mouse)

    • Left Click: Stamp shape to map
    • Scroll Wheel: Rotate heading
    • Ctrl + Scroll: Scale / Change Distance
    • Shift + Scroll: Native map zoom

    Drawing Controls (Keys)

    • Up / Down Arrows: Adjust Resolution Points (smoothness geometry)
    • Left / Right Arrows: Adjust Sweep Angle (Turns only. Negative flips left/right)
    • Escape: Cancel drawing mode
