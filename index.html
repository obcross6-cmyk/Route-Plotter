    🕹️ Planner Controls & Changelog

    Basic Mapping & Paster

    • Left Click Map: Place waypoint
    • Middle Click Map: Drop waypoint freely (even with ruler/compass open)
    • Right Click Waypoint: Open context menu OR snap active Compass/Ruler start
    • Paste Route String: Paste DDMMSS coordinates (e.g. 530642N0003943W) and press Enter to import route!

    Drawing Setup & Direct Radial

    • Start Drawing: Right-click waypoint -> Select mode or Direct Radial
    • Anchoring: Shapes automatically stick to the selected waypoint.
    • Backspace / Delete: Detach shape from waypoint so it follows mouse freely.
    • Tab Key: Lock heading angle in Direct Radial mode.

    Drawing Controls (Mouse)

    • Left Click: Stamp shape to map
    • Scroll Wheel: Rotate heading
    • Ctrl + Scroll: Scale / Change Distance
