    🕹️ Planner Controls

    Basic Mapping

    • Left Click Map: Place waypoint
    • Middle Click Map: Drop waypoint freely (even while measuring)
    • Right Click Waypoint: Open context menu OR snap active Compass/Ruler
    • Drag Map: Pan view
    • Paste Route: Paste coordinate strings (e.g. 530642N0003943W) to import!

    Drawing Setup

    • Start Drawing: Right-click waypoint -> Select mode
    • Anchoring: Shapes automatically stick to the selected waypoint.
    • Backspace / Delete: Detach shape from waypoint so it follows mouse freely.
    • Direct Radial (Tab): Press Tab to lock heading. Arrow keys adjust.

    Drawing Controls (Mouse)

    • Left Click: Stamp shape to map
    • Scroll Wheel: Rotate heading
    • Ctrl + Scroll: Scale / Change Distance
    • Shift + Scroll: Zoom map in/out

    Drawing Controls (Keys)

    • Up / Down Arrows: Fine-tune Resolution Points by ±1 point (or adjust Radial Angle)
    • Left / Right Arrows: Adjust Sweep Angle (Turns only) or adjust Radial Angle.
    • Escape: Cancel drawing mode
